### Requirements
  * Node.js 16.0 or above

### How to run
  * Clone or download repository
  ```
  > git clone https://github.com/jungbomp/genie-console.git
  ```
  * Install packages
  ```
  > npm ci
  ```
  * Run
    * debug mode
    ```
    > npm run start
    ```
  * Build
    ```
    > npm run build
    ```
    
### Default url
  * localhost:8000
